# TraceGuard
Detect modern C2 evasion techniques: DNS tunneling, Fast Flux, DGA, SSL/TLS C2, and cloud abuse (GitHub/Firebase). MITRE ATT&amp;CK-aligned. Python-based, Zeek log compatible. For SOC &amp; threat hunters. 检测现代C2隐蔽技术：DNS隧道、快速切换DNS（Fast Flux）、域名生成算法（DGA）、SSL/TLS C2及云服务滥用（GitHub/Firebase）。符合MITRE ATT&amp;CK框架，基于Python，兼容Zeek日志，专为安全运营中心（SOC）与威胁猎人设计。
